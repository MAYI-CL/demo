1.模板中的变量输出

  • {{ name }} {{ age }}
  • {{stu.name}} {{stu.age}}
  • {{a.0}} {{a.1}} {{a.2}}

2.模板中的标签使用

{% for vo in dlist %} {% endfor %} 令牌：{% csrf_token %}
姓名 性别 年龄 状态
{{vo.name}} {% if vo.sex == 1 %}男{% else %}女{% endif %} {{vo.age}} {% if vo.state == 0 %} 管理员 {% elif vo.state == 1 %} VIP账户 {% else %} 禁用账户 {% endif %}

3.过滤器的使用

  • 原值：{{ name }}
  • 大写：{{ name|upper }}
  • 小写：{{ name|lower }}
  • {{time}}
  • {{time|date:'Y-m-d H:i:s w'}}
  • {{a|join:':'}}

4.模板中的运算

  • 二个数：{{m1}},{{m2}}
  • 求和：{{ m1|add:m2}}
  • 求差：{{ m1|add:-20}}
  • 相乘：{% widthratio m1 1 m2 %}
  • 相除：{% widthratio m1 m2 1 %}

自定义过滤器和模板标签

{% load pagetag %} 大写：{{name|myupper}}
